# CRUD Elegante

Projeto desenvolvido em ASP.NET MVC utilizando Dapper e PostgreSQL para realizar operações CRUD (Create, Read, Update e Delete).

---

# Funcionalidades

- Cadastro de usuários
- Listagem de registros
- Atualização de dados
- Exclusão de informações
- Integração com banco PostgreSQL
- Interface simples e organizada

---

# Tecnologias Utilizadas

- C#
- ASP.NET MVC
- Dapper
- PostgreSQL
- GitHub
- HTML/CSS

---

# Desenvolvimento Colaborativo

Projeto realizado em equipe com foco em organização, separação de responsabilidades e boas práticas no desenvolvimento.
